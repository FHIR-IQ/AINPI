# Code of Conduct

This project adopts the **Contributor Covenant 2.1** verbatim. The canonical text lives at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Scope

The Code of Conduct applies to all project spaces: the GitHub repository (issues, PRs, discussions), sibling repositories under the `FHIR-IQ` organization, and any project-branded communication channel.

## Enforcement

Report violations privately to **gene@fhiriq.com**. Reports will be reviewed within 48 hours. Project maintainers who violate the Code of Conduct may face temporary or permanent consequences as outlined in the Contributor Covenant.
